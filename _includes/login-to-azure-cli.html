{% capture az-login %} In your terminal, run the following command: ```console az login ``` A browser window opens at `https://login.microsoftonline.com/organizations/oauth2/v2.0/authorize`. Select your Microsoft account to log in. If a window does not open, cancel the command (**Ctrl + C**) and run the following command instead: ```console az login --use-device-code ``` Then follow the directions in the terminal to log in. {% endcapture %}
