1. Go to https://portal.azure.com.
  2. Select Create one! {% include figure popup=true image_path="/assets/images/create-azure-account-start.png" alt="Create new Azure account link" %}
  3. Enter your email address and then select Next.
  4. Enter a password and then select Next. {%- capture password-notice -%}

    Passwords must have at least 8 characters and contain two of the following:

    • uppercase letters
    • lowercase letters
    • numbers
    • symbols
    {%- endcapture -%} {% include primary-notice.html content=password-notice %}

  5. Enter your country/region and birthdate. Then select Next.

    Azure sends you a verification email to the address you entered earlier.

  6. Enter the six-digit code from the verification email and then select Next.

    {%- capture tip-content -%} Select the checkbox next to I would like information, tips, and offers about Microsoft products and services to unmark that option. This tells Microsoft to not send you marketing emails. {%- endcapture -%} {% include tip-notice.html content=tip-content %}

  7. Follow the instructions to prove you're not a robot.

You'll see the Azure homepage after you've successfully created your account. {% include figure popup=true image_path="/assets/images/create-azure-account-finish.png" alt="Complete Azure account creation" %}
